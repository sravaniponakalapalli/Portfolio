Sentiment Analysis on LinkedIn App Reviews

Executive Summary

This project applies Natural Language Processing (NLP) techniques to analyze user reviews of the LinkedIn mobile application. Using TextBlob-based sentiment analysis, reviews are automatically classified as Positive, Negative, or Neutral. The analysis helps uncover customer perception, common issues, and overall satisfaction trends, enabling data-driven product and business decisions.

Business Problem

Organizations receive large volumes of unstructured textual feedback through app reviews and social media. Manual sentiment analysis is time-consuming, inconsistent, and not scalable. There is a need for an automated solution to understand customer sentiment, monitor brand reputation, and identify recurring pain points efficiently.

Objective

  • Automatically classify customer reviews into Positive, Negative, and Neutral sentiments
  • Analyze the relationship between user ratings and sentiment
  • Explore review length patterns and sentiment distribution
  • Extract frequently occurring words driving customer opinions

Dataset Overview

  • Source: LinkedIn App Reviews (CSV file)
  • Total Records: 702 reviews
  • Key Columns: Review (text), Rating (1–5)

Data Preparation & Exploration

  • Loaded and cleaned review data using Pandas
  • Verified data types and completeness
  • Created a new feature to capture review length
  • Explored rating distribution and review length patterns

Sentiment Analysis Methodology

Sentiment analysis was performed using the TextBlob NLP library. Each review was analyzed for sentiment polarity and classified as:

  • Positive: Polarity > 0.1
  • Negative: Polarity < -0.1
  • Neutral: Polarity between -0.1 and 0.1

Dashboard Features & Visual Analysis

Distribution of Ratings

  • Visualizes how user ratings are distributed across the dataset
  • Helps identify rating skewness and dissatisfaction trends

Review Length Analysis

  • Shows the distribution of review lengths
  • Longer reviews tend to contain stronger or more detailed sentiment

Overall Sentiment Distribution

  • Displays the count of Positive, Negative, and Neutral reviews
  • Negative sentiment dominates, aligning with lower user ratings

Sentiment vs Rating

  • Compares sentiment categories across different rating levels
  • Lower ratings strongly correlate with Negative sentiment

Word Cloud Analysis

  • Generated separate word clouds for Positive, Negative, and Neutral reviews
  • Highlights recurring keywords driving customer opinions
  • Helps identify common complaints, strengths, and feature requests

Key Insights

  • Strong alignment between star ratings and sentiment polarity
  • Negative reviews dominate lower ratings, validating sentiment logic
  • Word clouds reveal actionable feedback themes
  • Automated NLP significantly reduces manual review effort

Tools & Technologies Used

  • Python
  • Pandas & NumPy
  • Matplotlib & Seaborn
  • TextBlob (NLP & Sentiment Analysis)
  • WordCloud
  • Jupyter Notebook

Conclusion

This project demonstrates how NLP techniques can be used to extract meaningful insights from unstructured customer feedback. By automating sentiment analysis, businesses can better understand user perception, identify product issues, and make informed, data-driven decisions. The approach can be extended to other platforms such as social media, e-commerce reviews, and customer support data.
